# أفكار وتحسينات للمشروع

استخدم هذا الملف لتسجيل أفكار جديدة أو تحسينات. انسخ القالب من أسفل الصفحة لكل فكرة.

---

## تدقيق الأقسام — منتج للمحلات (نقلة نوعية)

التطبيق منتج يُباع لمحلات وتجار. تم تدقيق **كل قسم** في البرنامج وملؤه باحتياجات التاجر وأفكار عالمية.

- **المستند الكامل:** [`docs/MERCHANT_SECTIONS_SPEC.md`](docs/MERCHANT_SECTIONS_SPEC.md)
- **أولويات:** P0 (لا بد منها) → P1 (تجربة تاجر قوية) → P2 (تميز ونقلة نوعية)
- **أهم النواقص الحالية:** بحث موحّد، تنبيه مخزون منخفض، مرتجع مشتريات في القائمة، موردين، ترقيم وطباعة سندات، لوحة تحكم (مبيعات اليوم + رصيد صندوق)، جرد مخزون، تصدير تقارير PDF/Excel، "عن البرنامج"

---

## القالب (انسخه لكل فكرة جديدة)

```markdown
### [اسم الفكرة]
- **الوصف:** ...
- **الهدف / الفائدة:** ...
- **أين:** واجهة / API / كلاهما
- **الأولوية:** عالية / متوسطة / منخفضة
- **الحالة:** لم تبدأ | قيد التنفيذ | منتهية
- **ملاحظات:** ...
```

## أفكار مقترحة (أمثلة — عدّلها أو احذفها)

### إشعارات للمخزون المنخفض
- **الوصف:** تنبيه عند وصول صنف تحت حد معين (حد أدنى للمخزون).
- **الهدف:** تجنب نفاد الأصناف دون إنذار.
- **أين:** API (إعدادات حد أدنى + نقطة واحدة للتحقق)، واجهة (بطاقة أو تنبيه في لوحة التحكم).
- **الأولوية:** متوسطة
- **الحالة:** لم تبدأ

### تقرير المبيعات اليومي (PDF أو طباعة)
- **الوصف:** تقرير يومي بالمبيعات والإيرادات مع إمكانية الطباعة أو التصدير.
- **الهدف:** متابعة يومية سريعة.
- **أين:** API (تجميع بيانات من الفواتير)، واجهة (زر طباعة/تصدير).
- **الأولوية:** متوسطة
- **الحالة:** لم تبدأ

### دعم الباركود من الكاميرا
- **الوصف:** مسح الباركود من كاميرا الجوال بدل الإدخال اليدوي.
- **الهدف:** تسريع نقطة البيع على الموبايل.
- **أين:** واجهة فقط (استخدام API الكاميرا + مكتبة قراءة باركود).
- **الأولوية:** منخفضة
- **الحالة:** لم تبدأ

## أفكار مبهرة (Wow) — مرتبطة بتدقيق الأقسام (لا تكرار)

الأفكار التالية **تكمل** الأقسام في [docs/MERCHANT_SECTIONS_SPEC.md](docs/MERCHANT_SECTIONS_SPEC.md) و [docs/GLOBAL_ACCOUNTING_ROADMAP.md](docs/GLOBAL_ACCOUNTING_ROADMAP.md). لا تُعتبر مفاهيم جديدة منفصلة؛ الرجوع للقسم "أفكار مبهرة (Wow) — ربط مع الأقسام أعلاه" في MERCHANT_SECTIONS لتفاصيل الربط.

| الفكرة | يكمّل |
|--------|--------|
| اختصارات لوحة المفاتيح (F2، Ctrl+K، …) | MERCHANT: التعليمات — عن البرنامج |
| دفع نقدي سريع + اقتراح سعر + تحذير تحت التكلفة + تنبيه دين العميل | MERCHANT: المبيعات — فاتورة بيع |
| مبيعات اليوم + تذكير فواتير معلقة على اللوحة | MERCHANT: ملف — لوحة التحكم |
| تنبيه مخزون منخفض | MERCHANT: المستودع — بطاقة مادة، جرد؛ نفس مفهوم "إشعارات للمخزون المنخفض" أعلاه |
| تصدير PDF/Excel + لوحة KPIs | GLOBAL_ACCOUNTING_ROADMAP مرحلة ٦؛ MERCHANT: مركز التقارير |
| ترقيم وطباعة إيصال قبض/صرف | MERCHANT: السندات |
| فاتورة بالعملة + سعر صرف يدوي + لغة تقارير + قوالب دليل حسابات | GLOBAL_ACCOUNTING_ROADMAP مراحل ١ و ٧ |

*أضف أفكارك تحت هذا السطر باستخدام نفس القالب.*
